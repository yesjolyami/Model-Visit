# Namitova-K Portfolio

Профессиональное портфолио модели с современным дизайном и многоязычной поддержкой.

## 🏗️ Архитектура проекта

### Структура компонентов

```
src/
├── components/
│   ├── common/          # Общие компоненты
│   │   ├── Preloader.jsx/css
│   │   └── Footer.jsx/css
│   ├── layout/          # Компоненты макета
│   │   └── Background.jsx/css
│   ├── ui/              # UI компоненты
│   │   ├── GlassCard.jsx/css
│   │   ├── LanguageToggle.jsx/css
│   │   └── Lightbox.jsx/css
│   ├── features/        # Функциональные компоненты
│   │   ├── GalleryHeader.jsx/css
│   │   └── GalleryContent.jsx/css
│   └── index.js         # Экспорт всех компонентов
├── hooks/
│   └── useLoading.js    # Кастомные хуки
├── utils/               # Утилиты
├── locales/             # Файлы переводов
│   ├── en.json
│   └── ru.json
├── i18n.js              # Конфигурация интернационализации
├── App.jsx/css          # Главный компонент
├── Gallery.jsx/css      # Страница галереи
└── main.jsx             # Точка входа
```

### Ключевые особенности

- **Компонентная архитектура**: Код разбит на переиспользуемые компоненты
- **Многоязычная поддержка**: i18next с английским и русским языками
- **Адаптивный дизайн**: Responsive layout для всех устройств
- **Производительность**: Ленивая загрузка изображений, оптимизированные анимации
- **Доступность**: Семантическая разметка, клавиатурная навигация

### Используемые технологии

- **React 19** - Основной фреймворк
- **i18next** - Интернационализация
- **CSS Modules** - Стилизация компонентов
- **React Icons** - Иконки
- **Vite** - Сборщик проекта

### Запуск проекта

```bash
npm install
npm run dev
```

### Структура состояний

- **App**: Управление страницами и глобальным состоянием
- **Gallery**: Управление активными табами и лайтбоксом
- **Background**: Автоматическая ротация фоновых изображений
- **LanguageToggle**: Переключение языков с сохранением в localStorage
